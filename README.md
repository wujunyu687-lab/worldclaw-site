# WorldClaw Site

This repository hosts the public GitHub Pages site for **WorldClaw**.

It is intentionally separate from the working implementation repository.

## What lives here

- `index.html`
- `site/styles.css`
- `site/main.js`

This repository is for the public project overview only.

## What does not live here

- training code
- evaluation code
- bot runtime code
- local configuration
- credentials
- private project documents

## Site URL

- `https://wujunyu687-lab.github.io/worldclaw-site/`

## Purpose

The public site explains:

- project scope
- platform entry surfaces
- bot topology
- module responsibilities
- high-level workflow

The actual working codebase can remain private.
